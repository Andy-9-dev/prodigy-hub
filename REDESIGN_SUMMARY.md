## DESIGN SYSTEM SPECIFICATIONS

### Color Palette
```
PRIMARY COLORS
• Crimson Red (#C0392B) — Main brand color, CTA buttons
• Dark Red (#922B21) — Hover states
• Deep Crimson (#7B241C) — Footer background

SECONDARY COLORS
• Light Sky Blue (#EBF5FB) — Main background
• Pale Ice Blue (#D6EAF8) — Section backgrounds
• Soft Blue (#AED6F1) — Borders

ACCENT COLORS
• Warm Gold (#F39C12) — Stars, badges, highlights
• Soft Red (#F9EBEA) — Icon backgrounds
• Light Pink (#FADBD8) — Footer text

TEXT COLORS
• Deep Navy (#1A1A2E) — Headings
• Medium Slate (#4A4A68) — Body text
```

### Typography
- Display/Headings: Playfair Display (added for authority)
- Body: Inter (clean, readable)
- Font Weights: 400, 500, 600, 700

### Spacing & Radius
- Border Radius: 6px (sm), 8px (md), 12px (lg)
- Section Padding: 72px (desktop), 56px (tablet)
- Transition: 250ms ease (all interactive elements)

## SUPPORT & MAINTENANCE

### Image URLs
All images are hosted on Unsplash CDN (free, reliable, unlimited bandwidth)
- Format: `https://images.unsplash.com/photo-[ID]?w=[WIDTH]&q=80`
- High quality, automatically optimized
- No licensing concerns (Unsplash free license)

### CSS Customization
All colors are defined as CSS custom properties in `:root {}`, making future updates trivial:
```css
--accent: #C0392B;  /* Change this one line to update brand color everywhere */
```

### Performance
- Minimal CSS (5.88 kB gzipped)
- Efficient image URLs with width parameters
- No unnecessary JavaScript
- Fast-loading optimized build
